## 💡 About

A dead simple Wordle game for your command-line that you can choose custom word or length.

## 📥 Installation

Add this line to your application's Gemfile:

```ruby
gem 'wordle_cli'
```

And then execute:

```sh
bundle install
```

Or install it yourself as:

```sh
gem install wordle_cli
```
